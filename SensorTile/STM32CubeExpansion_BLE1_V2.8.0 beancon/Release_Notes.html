Release Notes for X-CUBE-BLE1

Software for STM32, Expansion for STM32Cube

Copyright 2016 STMicroelectronics

 


The X-CUBE-BLE1 software package is an expansion for STM32Cube. This software provides drivers running on STM32 for STM's BlueNRG/BlueNRG-MS Bluetooth Low Energy device. It includes many sample applications. This software package is built on top of STM32Cube software technology that ease portability across different STM32 microcontrollers.

X-CUBE-BLE1 software features:
  • Complete middleware to build applications using BlueNRG/BlueNRG-MS network processor.
  • Easy portability across different MCU families thanks to STM32Cube.
  • Sample applications that the developer can use to start experimenting with the code.
  • References to free Android and iOS Apps that can be used along with the sample applications.
  • Free user-friendly license terms.
  • Examples implementation available for X-NUCLEO-IDB04A1/X-NUCLEO-IDB05A1 STM32 expansion board plugged on top of one NUCLEO-F401RE, NUCLEO-F411RE, NUCLEO-L053R8 or NUCLEO-L476RG.

The figure below shows the overall architecture.
     
         

  • At the bottom layer there are the HW components: the STM32 MCU and the BlueNRG/BlueNRG-MS network processor.

  • The drivers abstract low level details of the hardware and allow the middleware software to provide Bluetooth LE features in a hardware independent fashion.
  • The applications provide examples of how to use the code.
